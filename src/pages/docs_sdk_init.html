Docs : SDK : init

init lets you initialize your Userbase client and automatically log in a user using the last used session.

    userbase.init({ appId })
      .then((session) => {
        // Userbase client initialized successfully
        
        if (session.user) {
          // there is a valid active session
          console.log(session.user.username)
        }
      })
      .catch((e) => console.error(e))

Parameters

  • appId [string] - Your App ID you received when you created a Userbase application.

Result

  • session [object] - Contains information about the last used session.
    • user [object] - If set, this contains information about the logged in user.
      • username [string] - The user's username.
      • key [string] - The user's encryption key.
      • email [string] - The user's email.
      • profile [object] - The user's profile.
    • lastUsedUsername [string] - If set, this is the username of the last user that was logged in.

Errors

  • AppIdMustBeString
  • AppIdCannotBeBlank
  • AppIdNotValid
  • KeyNotFoundHandlerMustBeFunction
  • UserAlreadySignedIn
  • UserCanceledSignIn
  • ServiceUnavailable
