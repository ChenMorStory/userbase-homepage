SDK : forgotPassword

forgotPassword lets you send a temporary password to a user's email. This API will return a promise that gets resolved once the email with the temporary password has been sent.

    userbase.forgotPassword(username)
      .then(() => {
        // email with temporary password sent
      })
      .catch((e) => console.error(e))

Parameters

  • username [string | Len: 1-50] - The username for the account to send a temporary password to.

Errors

  • UsernameCannotBeBlank
  • UsernameMustBeString
  • AppIdNotSet
  • AppIdNotValid
  • UserNotFound
  • UserEmailNotFound
  • ServiceUnavailable
