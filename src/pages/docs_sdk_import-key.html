SDK : importKey

importKey lets you manually import a user's key. The user must be signed in to import their key. This API will return a promise that gets resolved once the key has been imported successfully.

    userbase.importKey(key)
  .then(() => {
    // key imported successfully
  })
  .catch((e) => console.error(e))

Parameters

  • key [string] - The user's secret key.

Errors

  • KeyMustBeString
  • KeyCannotBeBlank
  • KeyNotValid
  • UserNotSignedIn
  • ServiceUnavailable
