SDK : Configure

Configure lets you configure your Userbase client.

    userbase.configure({ appId, endpoint })

Parameters

  • settings [object] - Contains your configuration settings.
    • appId [string] - Your App ID you received when you created a Userbase application.
    • endpoint [string] - If you're hosting your own Userbase server, you can provide its endpoint here.

Errors

  • ConfigParametersMissing
  • UserAlreadySignedIn
