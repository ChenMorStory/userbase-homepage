SDK : SignUp

SignUp lets you create user accounts for your web app. This API with will return a promise that gets resolved once the user account has been created.

    userbase.signUp(username, password)
  .then((user) => {
    // user account created
  })
  .catch((e) => console.error(e))

Parameters

  • username [string | Len: 1-50] - The username for the account to be created.
  • password [string | Len: 6-1000] - The password for the account to be created.

Result

  • user [object] - Contains information about the new user.
    • username [string] - The user's username.
    • key [string] - The user's encryption key.

Errors

  • UsernameAlreadyExists
  • UsernameCannotBeBlank
  • UsernameTooLong
  • UsernameMustBeString
  • PasswordCannotBeBlank
  • PasswordTooShort
  • PasswordTooLong
  • PasswordMustBeString
  • AppIdNotSet
  • AppIdNotValid
  • ServiceUnavailable
