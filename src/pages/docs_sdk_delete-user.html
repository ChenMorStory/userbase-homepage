Docs : SDK : deleteUser

deleteUser lets you delete a user and all their data from the server. This API will return a promise that gets resolved once the user account has been marked for deletion.

    userbase.deleteUser()
      .then((user) => {
        // user marked for deletion on the server
      })
      .catch((e) => console.error(e))

Errors

  • UserNotSignedIn
  • UserNotFound
  • ServiceUnavailable
