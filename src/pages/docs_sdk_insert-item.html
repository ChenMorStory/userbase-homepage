SDK : insertItem

insertItem lets you add items to a user's database. This API will return a promise that gets resolved once the item has been durably inserted into the database.

    userbase.insertItem(databaseName, item, itemId)
  .then(() => {
    // item inserted
  })
  .catch((e) => console.error(e))

Parameters

  • databaseName [string | Len: 1-50] - The database name to use.
  • item [object | Max size: 10 KB] - The item to insert.
  • itemId [string | Len: 1-100 | optional] - The item's unique identifier.

Errors

  • DatabaseNotOpen
  • DatabaseNameCannotBeBlank
  • DatabaseNameTooLong
  • DatabaseNameMustBeString
  • ItemIdTooLong
  • ItemIdMustBeString
  • ItemMissing
  • ItemTooLarge
  • ItemAlreadyExists
  • UserNotSignedIn
  • ServiceUnavailable
